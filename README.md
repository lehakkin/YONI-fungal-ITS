# YONI-fungal-ITS
Anlaysis of fungal ITS2 MiSeq sequences from four differently managed soils, forest, meadow, organic cropping system and conventional cropping system, and five soil layers, 0-10 cm, 10-20 cm, 20-30 cm, 30-40 cm, and 40-80 cm. Analysis include alpha and beta diversity, Principal coordinates analysis (PCoA) ,  PERMANOVA, composition barplots etc.
